Flask Magic Eightball
=====================

Note you must be inside the `src/` directory to run the app.

Run using Linux/Mac::

    $ export FLASK_APP=flask_app
    $ flask run

Run using Windows CMD::

    > set FLASK_APP=flask_app
    > flask run

Run using Powershell::

    > $env:FLASK_APP = "flask_app"
    > flask run
